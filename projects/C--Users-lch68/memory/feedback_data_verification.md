차트/시각자료 생성 시 데이터를 그냥 믿고 쓰면 안 됨. 반드시 검증 절차 거칠 것.

**Why:** 전세 토론 차트 세션에서 찬희님이 "진짜 데이터 맞냐?" 질문. 검증 결과 대부분 정확했지만, RIR 출처가 부정확(가계동향조사→주거실태조사), HUG 사고 수치가 시점별로 달랐음(72.3%→78%). 검증 안 했으면 잘못된 출처로 토론에 나갈 뻔함.

**How to apply:**
1. 사용자가 데이터를 제공해도 "검증 단계"를 반드시 넣을 것
2. nexus-finance MCP → WebSearch → WebFetch 3중 폴백
3. 검증 결과를 테이블로 정리 (수치 / 검증치 / 판정 / 출처URL)
4. 차트 출처에 "기관명 + 보고서명 + 발행일" 구체적으로
5. 검증 불가 항목은 사용자에게 명시적으로 고지
