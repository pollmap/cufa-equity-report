CUFA 기업분석보고서의 다크테마 인터랙티브 HTML 레이아웃이 검증됨. python-docx Word 파일 없이 HTML만으로도 충분.

**Why:** 유저가 "지금 레이아웃은 매우 마음에 든다", "굳이 워드 파일이 아니어도 지금 같은 형식은 매우 마음에 든다"라고 확인.

**How to apply:**
1. CUFA 보고서는 다크테마 HTML(.html) 단독으로 생성해도 됨. Word는 선택사항.
2. 텍스트 분량은 현재의 5배 이상 — 산업 전체, 경쟁사 상세, 당사 비즈니스 모든 세그먼트, 현황과 전망을 빠짐없이 커버.
3. 시각자료 대폭 확충 필수: 원그래프(pie), 선그래프(line), 바차트(bar), 워터폴, 산점도 등 SMIC/애널리스트 보고서 수준.
4. "이 파일 하나로 산업~경쟁사~당사 모든 것을 파악 가능" 수준이 기준선.
5. 레이아웃(다크bg, 퍼플 액센트, 사이드바+본문 2열, 메트릭카드, 시나리오카드 등) 유지.
