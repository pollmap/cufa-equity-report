**How to apply:**
1. Phase 0에서 DART 사업보고서 본문 정독 필수 (거래처/항로/선대/리스크)

3. 지정학 리스크 자동 스캔 (뉴스 검색 → 항로 교차확인)
4. 서브에이전트 병렬 빌드 (sections/*.py 파일 분리)
5. add_source 중복 방지 (chart_with_context 후 별도 호출 금지)
6. SVG auto_base (막대 비례 개선)
7. 기업 홈페이지 이미지만 사용 (스톡 이미지 금지)
8. Float TOC 겹침 방지 (max-width 960px)
9. 도입부 서사 규칙 (AI 문체 방지, 장면/비유/질문으로 시작)
10. 시각 자료 다양성 (차트+사진+지도+다이어그램 최소 5유형)
11. 레이더 차트 정량 데이터만 (CUFA 자체 정성 점수 금지)
12. expand_card 핵심 논거에 사용 금지
