**Why:** 123개 MCP 도구가 한국 금융 데이터 시장의 유일한 통합 서버. ECOS/KOSIS/BOK MCP = 0개 공백.

**How to apply:**
- 3단계: 에이전시(크몽/위시켓) → 프로덕타이즈드 서비스 → SaaS
- 수익화: MCPize(85% 분배), Apify(pay-per-event), x402(마이크로페이먼트)
- 가격: Free 100calls/day, Starter $49, Pro $149, Enterprise $499
- 정부지원: 예비창업패키지(최대 1억), 창업중심대학 — 사업자등록 전 신청
- 규제: NFA 원칙 유지, 데이터만 제공, 투자판단 조언 금지
- 해외 매출: 부가세 영세율(0%) + 매입세액 환급
- 가상자산: 2027.01.01까지 비과세
- 벤치마크: SiteGPT(24세 인도, $1.14M ARR), AI SaaS 중위 11개월 to $1M ARR
