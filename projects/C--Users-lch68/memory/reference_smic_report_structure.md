## 표준 섹션 구조
1. 표지 (종목명, Rating, 목표주가, Key Metrics, 팀원, 캐치프레이즈 제목)
2. 목차 (CONTENTS)
3. 산업분석 / 기업분석
4. 주가분석
5. 투자포인트 (2~3개, 각 5~10p)
6. ISSUE & RISK
7. 매출 추정 (사업부문별 Bottom-up)
8. Valuation (방법론 선택 근거 필수)
9. Appendix (BS/CF 필수, 추정 근거 테이블)

## Valuation 방법론 선택 패턴
- 바이오 -> rNPV 또는 DCF (임상 확률 가중)
- 지주사 -> SOTP (EV/EBITDA 기반)
- 성장주 -> Historical PER
- Base Case / Bull Case 시나리오 분석 일반적
