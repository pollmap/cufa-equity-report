스크롤 유지하되 SMIC처럼 각 섹션이 "페이지처럼" 보이는 구조. 다크 테마 전용 (화이트 토글 삭제).

**Why:** 스크롤 방식은 시각 자료가 본문 중간에 끼어서 가독성 나쁨. SMIC 보고서는 페이지 단위로 텍스트+차트가 정돈되어 있음. 찬희님 "시각자료가 중간에 나오지말고 한페이지한페이지씩 하는걸로"

**How to apply:**
- build_template.py gen_css()에 CSS paged media 또는 JS 페이지네이션 구현
- 각 페이지: 고정 높이 (A4 비율 or 화면 높이)
- 페이지 내에서 텍스트+차트 배치 (차트가 페이지를 넘지 않도록)
- 좌우 화살표 또는 스와이프로 페이지 전환
- 페이지 번호 표시
- 인쇄 시 자연스럽게 페이지 분리
- Float TOC → 페이지 번호 기반으로 변경

JYP 보고서 v1 추가 문제점:
- SVG 차트 배경 다크테마 불일치 (흰배경 산점도/워터폴)
- 막대 크기 너무 작음 (auto_base 미적용)
- 레이더 자체평가 금지 규칙 무시
- sidebar_wrap 2열 레이아웃 제대로 안 됨
- 기업 이미지 0개
- 서브에이전트에 build_template.py 사용법 상세 전달 필요
