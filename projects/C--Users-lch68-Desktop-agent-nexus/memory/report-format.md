# Investment Report Format (CUFA Standard)

## Report Types
| Type | Pages | Purpose | When |
|------|-------|---------|------|
| 기업분석 보고서 | 30-60p | 신규 커버리지 개시 | 처음 분석 시 |
| 업데이트 보고서 | 5-15p | 이벤트 후 의견 갱신 | 분기 실적 후 |
| 긴급 보고서 | 1-3p | 돌발 이벤트 대응 | M&A, 규제변경, 사고 |
| 산업 보고서 | 20-50p | 산업 동향/전망 | 구조 변화 시 |
| 투자 메모 | 1-2p | 개인 투자 기록 | 매매 시 |

## Standard Report Structure (7 Sections)

### 1. Executive Summary (1 page)
- 기업명/종목코드, 투자의견(매수/보유/매도), 목표주가, 현재주가, 상승여력
- 핵심 투자 포인트 3줄
- 엘리베이터 피치: "[기업명]은 [핵심 사업] 기반으로 [성장 동인]이 기대됩니다. 현재 주가는 [지표] 기준 [매력도]이며, [촉매]를 고려할 때 [목표주가]까지 [상승여력]%의 상승 여력이 있습니다."

### 2. Investment Thesis (핵심 3가지)
- 성장 동력/가치 촉매
- 재무적 강점/개선점
- 밸류에이션 매력도
- 컨센서스 대비 나의 시각 비교표 포함

### 3. Business Analysis
- 비즈니스 모델, 매출 구성, 시장 규모, 경쟁 환경, Moat 분석

### 4. Financial Analysis
- 수익성(마진, ROE, ROIC), 성장성(CAGR), 안정성(부채비율), 효율성(CCC)

### 5. Valuation
- DCF: 핵심가정 + FCFF + 적정가치 + 민감도분석
- 상대가치: Peer 멀티플 + Target 멀티플 근거
- 시나리오별: Bull(25%)/Base(50%)/Bear(25%) + 확률가중평균
- Football Field Chart: 방법론별 적정가치 범위 시각화

### 6. Risk Factors
- 리스크 매트릭스 (확률 x 영향도)
- 시나리오별 리스크 분석
- 촉매 캘린더 (6-12개월 이벤트)

### 7. Earnings Estimates
- 매출, 영업이익, 순이익, EPS, PER, PBR, ROE, 배당수익률

## Korea-Specific Sections (Required)
- 지배구조 분석 (오너 리스크, 관계사 거래)
- 연결/별도 구분 명시
- 환율 영향 분석 (수출기업)
- DART 공시 이력
- 코리아 디스카운트 분석

## Personal Investment Memo (1-page)
1. 한 줄 요약
2. 핵심 투자 포인트 (3가지)
3. 적정가치 vs 현재주가 + 상승여력
4. 주요 리스크 (2가지)
5. 핵심 모니터링 지표
6. 매도 조건

## Monitoring
- 분기 실적 ±10% 이내?
- 투자 포인트 3가지 유효?
- 산업 환경 구조적 변화?
- 밸류에이션 매력도?
- 새 리스크?
- 확신 수준 변화?

## Confidence Levels
- 높음: 포인트 3개 유효 + 밸류에이션 매력 → 5-10%
- 중간: 일부 불확실 + 적정 → 2-5%
- 낮음: 관찰 + 추가확인 → 1-2%
