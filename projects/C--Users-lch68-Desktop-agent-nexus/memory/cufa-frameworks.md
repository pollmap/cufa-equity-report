## Industry KPIs by Sector
- 반도체: ASP, 가동률, 미세공정 노드
- 은행: NIM, NPL 비율, BIS 비율
- 통신: ARPU, 가입자 수, 해지율
- 유통: SSSG, GMV
- 건설: 수주잔고, 미분양률
- 바이오: 임상 단계별 파이프라인 수
- 정유: 정제마진, 가동률
- 항공: Load Factor, RASK

## Valuation by Sector
- IT/반도체: EV/EBITDA (감가상각 차이, 사이클)
- 바이오: DCF Pipeline (적자 다수, 미래가치)
- 은행: PBR (레버리지, 자산 기반)
- 유틸리티: PBR + 배당수익률 (안정적 CF, 규제)
- 부동산/리츠: NAV + 배당수익률 (자산가치)
- 건설: PER 수주기반 (수주잔고→미래매출)
- 소비재: PER (안정적 이익)

## Analysis Frameworks
1. Porter's Five Forces: 경쟁강도, 신규진입, 대체재, 공급자교섭, 구매자교섭
2. Industry Lifecycle: 도입기→성장기→성숙기→쇠퇴기
3. Company Analysis 5-Step: 사업이해→경쟁우위→재무분석→밸류에이션→투자결정
4. Economic Moat: 네트워크효과, 전환비용, 비용우위, 무형자산, 규모의경제
